# CodeAlpha_Web-Scrapping

## Project Overview

This project is a web scraping application developed using Python, BeautifulSoup, Requests, and Pandas. The project extracts quotes and author information from the Quotes to Scrape website and stores the collected data in a structured CSV format for further analysis.

## Website Used

https://quotes.toscrape.com

## Objectives

* Learn the fundamentals of web scraping
* Extract useful information from HTML pages
* Practice data collection and dataset creation
* Store scraped data into CSV files

## Tools & Technologies

* Python
* BeautifulSoup
* Requests
* Pandas
* Google Colab / Jupyter Notebook

## Features

* Scrapes quotes from multiple pages
* Extracts quote text and author names
* Stores data in CSV format
* Beginner-friendly implementation

## Dataset Information

The dataset contains:

* Quote Text
* Author Name

## Project Workflow

1. Send HTTP requests to the website
2. Parse HTML using BeautifulSoup
3. Extract quote information
4. Store data in a list
5. Convert data into a Pandas DataFrame
6. Export dataset as CSV

## Output

The project generates a CSV file containing all scraped quotes and authors.

Through this project, I learned:

* Data extraction using BeautifulSoup
* Data handling using Pandas
